# 小红书数据分析指令包

## 数据来源

### 手动记录
每次发布后记录：
- 发布时间
- 标题
- 内容类型
- 24h/48h/72h数据

### 数据模板
```
| 日期 | 标题 | 类型 | 24h点赞 | 24h收藏 | 24h评论 | 72h点赞 | 备注 |
|------|------|------|---------|---------|---------|---------|------|
```

---

## 核心指标解读

### 基础指标

| 指标 | 含义 | 基准线 |
|------|------|--------|
| 点赞 | 内容认可 | 行业平均1-3% |
| 收藏 | 实用价值 | >10%为优质 |
| 评论 | 互动深度 | >5%为高互动 |
| 分享 | 传播意愿 | >3%为爆款潜力 |

### 计算公式

```
收藏率 = 收藏数 / 曝光数
互动率 = (点赞+收藏+评论) / 曝光数
完播率 = 完播数 / 播放数（视频）
```

---

## 分析流程

### 步骤1：数据收集

提取最近10条笔记数据：
```
请提供最近10条笔记的数据：
1. 标题
2. 发布时间
3. 内容类型
4. 曝光量
5. 点赞/收藏/评论数
```

### 步骤2：横向对比

| 维度 | 分析点 |
|------|--------|
| 标题模式 | 哪类标题数据更好？ |
| 内容类型 | 干货/故事/测评，哪种表现更好？ |
| 发布时间 | 哪个时间段数据更高？ |
| 封面风格 | 什么风格点击率更高？ |

### 步骤3：纵向分析

观察单条笔记的数据曲线：
- 前2小时：判断冷启动表现
- 24小时：判断是否进入推荐池
- 72小时：判断长尾表现

### 步骤4：归因分析

**数据好的原因**：
- 标题钩子强
- 内容有实用价值
- 发布时间对
- 封面吸引人

**数据差的原因**：
- 标题平淡
- 内容同质化
- 发布时间不佳
- 封面信息不清晰

## 爆款案例分析

### 案例：路标笔记

**基准数据**：
- CTR：22.6%
- 完播率：36%

**成功因素拆解**：
1. 标题：数字+结果承诺
2. 内容：可操作的步骤
3. 结构：清单式呈现
4. 互动：结尾有提问

**可复用模式**：
- 选题方向：个人成长+方法论
- 内容结构：3-5个要点
- 文案风格：简洁+具体

## 注意事项

1. **样本量** — 至少10条笔记才有统计意义
2. **时间窗口** — 同一篇笔记，看72小时数据
3. **平台波动** — 注意平台算法调整影响
4. **季节性** — 某些内容有季节波动
