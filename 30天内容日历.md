# 30天内容日历模板

> 日更不烧脑 —— 每周1次选题+批量生产，其他6天只要排版发布。

---

## 周节奏

```
周一上午：选题 + 批量写文（3-4篇）
周一下午：生成封面提示词 + AI出图
周二-周日：每天发1条
周六：数据分析复盘
```

---

## 模板使用说明

| 字段 | 说明 |
|------|------|
| 日期 | 发布日 |
| 选题 | 一句话概括 |
| 类型 | 干货/故事/清单/产品 |
| 封面 | A人物/B对比/C纯文字/D场景 |
| 钩子 | 前3行开头 |
| 状态 | ⬜待生产 / ✍️已写 / 🎨已出图 / ✅已发 |

## 使用技巧

1. **同类型封面统一风格** — 粉丝能一眼认出你的笔记
2. **干货:故事:清单 = 4:2:1** — 保持这个比例，账号人设清晰
3. **每周至少1条故事型** — 人设立起来，转化率才能上去
4. **周三和周五晚8点发干货** — 这两个时段收藏率最高
5. **周末发清单/轻内容** — 用户休闲刷，转发率高
