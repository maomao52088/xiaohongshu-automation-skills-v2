# 30天内容日历模板 · 30-Day Content Calendar

> 日更不烧脑 —— 每周1次选题+批量生产，其他6天只要排版发布。
> Daily posting without daily planning — batch your topics once a week, publish the rest.

---

## 周节奏 · Weekly rhythm

```
周一上午 Monday AM：选题 + 批量写文（3-4篇）/ pick topics + batch-write 3-4 posts
周一下午 Monday PM：生成封面提示词 + AI出图 / generate cover prompts + AI image gen
周二-周日 Tue-Sun：每天发1条 / publish 1 per day
周六 Saturday：数据分析复盘 / data review
```

---

## 模板使用说明 · How to use this template

| 字段 Field | 说明 Description |
|------|------|
| 日期 Date | 发布日 / publish date |
| 选题 Topic | 一句话概括 / one-line summary |
| 类型 Type | 干货/how-to / 故事/story / 清单/list / 产品/product |
| 封面 Cover | A人物/face B对比/comparison C纯文字/text D场景/scene |
| 钩子 Hook | 前3行开头 / opening 3 lines |
| 状态 Status | 待生产/draft / 已写/written / 已出图/imaged / 已发/published |

## 使用技巧 · Pro tips

1. **同类型封面统一风格** — 粉丝能一眼认出你的笔记
   **Keep cover style consistent by type** — followers should recognize your posts instantly
2. **干货:故事:清单 = 4:2:1** — 保持这个比例，账号人设清晰
   **How-to : Story : List = 4:2:1** — this ratio keeps your account identity sharp
3. **每周至少1条故事型** — 人设立起来，转化率才能上去
   **At least 1 story per week** — personality drives conversion
4. **周三和周五晚8点发干货** — 这两个时段收藏率最高
   **Publish how-tos Wed & Fri 8PM** — highest save-rate windows
5. **周末发清单/轻内容** — 用户休闲刷，转发率高
   **Lists & light content on weekends** — casual scrolling, high share rate
