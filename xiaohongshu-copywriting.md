# 小红书文案生成器

## 输入要求

使用前提供：
- 选题/标题
- 内容类型（干货/故事/测评/清单）
- 目标受众
- 预期数据目标

## 步骤1: 选择文案模板

### 模板A：干货型（高收藏）

```
【标题】数字+结果承诺

开头（1-2句）：
- 痛点直击 OR 结果预告

正文：
1. 第一点（核心方法）
   - 具体操作
   - 案例/数据
2. 第二点
3. 第三点

结尾：
- 总结一句话
- 互动提问
```

### 模板B：故事型（高互动）

```
【标题】身份认同+转折

开头：
- 场景描述
- 冲突引入

正文：
- 过程展开（起承转合）
- 关键转折点
- 结果呈现

结尾：
- 感悟总结
- 引发共鸣
```

### 模板C：清单型（高转发）

```
【标题】数量+利益点

开头：
- 一句话概括价值

正文：
✅ 第一点
❌ 常见误区
💡 小技巧

✅ 第二点
...

结尾：
- 收藏提醒
- 补充邀请
```

## 步骤2: 填充内容

根据选题，填充模板：

**示例**：
```
选题："普通人如何利用AI做副业"
类型：干货型
目标：收藏率 > 30%

---

【标题】0成本用AI做副业，第3周我就赚了5400元

💡 很多人以为AI副业要学编程，其实不用。

分享我测试了3个月的方法：

1️⃣ 找对赛道
不要追风口，找"信息差+AI"的交叉点
我选的是：小红书内容代运营
→ 需求大，AI能大幅提效

2️⃣ 工具免费
Claude Code：内容生成
Canva：封面制作
Notion：项目管理
→ 全程0成本

3️⃣ 第一单怎么来
不发广告，先免费帮1-2个人做
做好了，他们会帮你介绍
→ 口碑比流量重要

📌 关键：别想太多，先跑起来

你觉得AI副业最难的是什么？评论区聊聊👇
```

## 步骤3: 检查清单

生成后必须检查：

- [ ] 标题是否有数字或对比？
- [ ] 开头是否3秒内抓住注意力？
- [ ] 正文是否有可操作的具体步骤？
- [ ] 结尾是否有互动引导？
- [ ] 全文是否控制在300-500字？
- [ ] emoji是否自然（每段1-2个）？
- [ ] 段落是否短小（每段2-3行）？

## 步骤4: A/B测试

同一选题生成2个版本：

| 版本 | 差异点 | 适用场景 |
|------|--------|----------|
| A版 | 数据开头 | 新账号 |
| B版 | 故事开头 | 有人设的账号 |

## 输出格式

```markdown
## 文案V1

【标题】...

正文...

---
预计数据：点赞___ 收藏___ 评论___
优化建议：...
```

## 注意事项

1. **拒绝标题党** — 内容要配得上标题
2. **拒绝堆砌emoji** — 每段最多2个
3. **拒绝假大空** — 每句话都要有信息量
4. **拒绝照搬** — 借鉴结构，不是抄袭内容
