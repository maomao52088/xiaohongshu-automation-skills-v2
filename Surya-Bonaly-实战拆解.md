```

Human story + contrast + AI industry parallel
```

- **Compelling subject**: Black female figure skater who broke through a white-dominated sport

## 2. 封面：类型C 纯文字排版 · Cover: Type C (text-only)

### 设计策略 · Design strategy

不露脸、不场景、纯冲击力文字。

No face, no scene. Pure text impact.

```
主标题 Main title：她做了评审最不想看到的事
副标题 Subtitle：然后拿了满分
背景 Background：深色渐变 + 冰刀纹理 / dark gradient + skate blade texture
```

### 为什么选类型C · Why Type C

- 不依赖个人形象（账号还在养人设阶段）
- No personal image dependency (account was still building identity)
- 纯文字在小红书feed中有差异感
- Text-only stands out in a feed full of selfies and product shots
- 好奇钩子 + 反转 = 强迫点开
- Curiosity hook + twist = compels the tap

### 提示词 · The prompt

```
Dark gradient background, ice blue to deep navy,
minimalist design with subtle ice skate blade silhouette,
centered bold text area taking 40% of image,
cinematic lighting, 4:5 aspect ratio,
editorial magazine cover style
```

## 3. 文案结构 · Copy structure

### 开头（前3行决定命运） · The hook (first 3 lines decide everything)

```
她做了一个连评审团都没想到的动作。
全场安静了3秒钟。
然后评委给了满分。
```

**为什么有效 · Why it works**：3句话制造信息缺口 → 用户必须点进来才能知道「什么动作」「为什么」。
3 lines create an information gap → reader MUST click to find out "what move" and "why."

### 中间（干货段） · The body

- 不讲 AI，讲故事
- Don't talk about AI — tell a story
- Bonaly 被低估 → 用小众技术突破 → 被重新评价
- Bonaly underestimated → broke through with niche technique → re-evaluated
- 暗线：AI从业者的处境和她一样
- Subtext: AI builders are in the same position she was

### 结尾（引导互动） · The closer

```
你做过什么「评审不喜欢但你知道对」的事？
评论区聊聊。
```

## 5. 复盘：为什么能跑出22.6% · Why it hit 22.6%

### 点击层（CTR） · Click layer

1. **封面好奇心** — 没见过的名字 + 反转标题 = 非点不可
   **Cover curiosity** — unfamiliar name + twist title = can't not click
2. **feed差异化** — 周围都是自拍/产品图，纯文字封面脱颖而出
   **Feed differentiation** — surrounded by selfies, text-only cover pops

### 内容层（收藏+评论） · Content layer (saves + comments)

1. **信息增量大** — 99%用户不知道这个人
   **High information novelty** — 99% of users didn't know this person
2. **可讨论性强** — 「评审不喜欢但正确」是普适情境
   **Discussion-worthy** — "doing what's right even when judges disagree" is universally relatable

### 传播层（分享） · Share layer

1. **体育+逆袭** — 天然传播基因
   **Sports + underdog** — built-in viral DNA
2. **无AI痕迹** — 传播时不带「卖课感」
   **Zero AI scent** — doesn't smell like a course pitch when shared
