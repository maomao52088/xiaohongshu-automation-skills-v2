## 2️⃣ 封面：类型C 纯文字排版

### 设计策略

不露脸、不场景、纯冲击力文字。

```
主标题：她做了评审最不想看到的事
副标题：然后拿了满分
背景：深色渐变 + 冰刀纹理
```

### 为什么选类型C

- 不依赖个人形象（账号还在养人设阶段）
- 纯文字在小红书feed中有差异感
- 好奇钩子 + 反转 = 强迫点开

### 提示词

```
Dark gradient background, ice blue to deep navy,
minimalist design with subtle ice skate blade silhouette,
centered bold text area taking 40% of image,
cinematic lighting, 4:5 aspect ratio,
editorial magazine cover style
```

## 3️⃣ 文案结构

### 开头（前3行决定命运）

```
她做了一个连评审团都没想到的动作。
全场安静了3秒钟。
然后评委给了满分。
```

**为什么有效**：3句话制造信息缺口 → 用户必须点进来才能知道「什么动作」「为什么」。

### 中间（干货段）

- 不讲 AI，讲故事
- Bonaly 被低估 → 用小众技术突破 → 被重新评价
- 暗线：AI从业者的处境和她一样

### 结尾（引导互动）

```
你做过什么「评审不喜欢但你知道对」的事？
评论区聊聊。
```

## 5️⃣ 复盘：为什么能跑出22.6%

### 点击层（CTR）

1. **封面好奇心** — 没见过的名字 + 反转标题 = 非点不可
2. **feed差异化** — 周围都是自拍/产品图，纯文字封面脱颖而出

### 内容层（收藏+评论）

1. **信息增量大** — 99%用户不知道这个人
2. **可讨论性强** — 「评审不喜欢但正确」是普适情境

### 传播层（分享）

1. **体育+逆袭** — 天然传播基因
2. **无AI痕迹** — 传播时不带「卖课感」
